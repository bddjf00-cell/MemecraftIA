# MemeCraft IA Code Assistant

Proyecto MemeCraft IA con implementaciones en múltiples lenguajes.

## Archivos

| Archivo | Lenguaje | Descripción |
|---------|----------|-------------|
| `index.html` | HTML/JS | Web UI con chat y carga de proyectos |
| `server.js` | Node.js | Servidor HTTP + proxy a NVIDIA NIM |
| `app.js` | Node.js | Chat CLI en Node.js |
| `Program.cs` | C# | Chat CLI en C# (.NET) |
| `script.py` | Python | Chat CLI en Python |
| `README.md` | Markdown | Este archivo |

## Cómo usar

### Web
```bash
node server.js
# Abre http://localhost:3000
```

### CLI Node.js
```bash
node app.js
```

### CLI Python
```bash
python script.py
```

### CLI C#
```bash
dotnet run Program.cs
```
